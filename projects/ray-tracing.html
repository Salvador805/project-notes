Ray Tracing on a Raspberry Pi

Introduction

Note that I made this as a final project for my physics lab. This was a very compact, info-heavy course, where we covered computer science, programming, networking, raster graphics, threading, forking and embedded systems. Note that there is still more to it, but those are all that I remember. This course was taken over summer 2024, in just 6 weeks. As if this wasn't already hard in a normal quarter system for someone who doesn't know much about computers, it was crunched in 6 weeks. I managed to also learn how to set up SSH connectivity for when I don't want to work on my desktop I can work from my laptop. For this project I had big help from a book called Ray Tracing in One Weekend by Peter Shirley, Trevor David Black, and Steve Hollasch. It was written in C, but I wrote my program in python, so I had to learn how to read C for this project. I also watched a couple of videos on ray tracing. All of it is in the references below.
