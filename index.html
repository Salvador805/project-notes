Ray Tracing on Raspberry Pi

Built a ray tracer with one basic shapes, a sphere and two basic materials(one metal and the other using lambertian diffusion). Tried adding a triangle as well and managed to get an output but there was some technical difficulties where I couldn't bring it in the frame, you would only be able to see it reflecting from the metal spheres as shown. The code was written in python and inspired from a book with code written in C.

Skills Worked: Python, Object Oriented Programming, Read C, Raspberry Pi OS, SSH Connectivity

AM Radio

Started this project around May 2024. Tried building an AM Radio on a breadboard using a wave function generator for the message wave, a 555 timer for the carrier wave, and an oscilloscope to troubleshoot. I also tried using a wave function generator for both carrier and message wave. I considered this a fail because it was part of a course and I never got to finish it. I revisited this project around January of 2026, because I resisted not knowing where I messed up. I took this opportunity to learn LTspice. I ended up replicating the circuit with a 555 timer for carrier and wave function generator for the message.

Skills Worked: Amplitude Modulation, Breadboard Prototyping, Oscilloscope, Analog Circuit Design, LTspice, Spice Directives, Troubleshooting
